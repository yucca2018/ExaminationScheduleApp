# ExaminationScheduleApp

![Image1](https://user-images.githubusercontent.com/98800328/161226029-6ba5385d-80db-444e-b820-10bb09ff8ce8.png)


### 概要
- 公務員試験の日程を確認するアプリです。
- 受験者数の多い国家公務員、地方公務員先の採用ホームページを掲載しています。
- 予定を確認後、予定表の作成ができます。

＊地方消防、市区町村（役所）に関しては政令指定都市のみの掲載です。
